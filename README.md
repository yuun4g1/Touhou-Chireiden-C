SCU-CCS 大一C语言课程期末游戏项目 《东方地灵殿》
操作方式：
  主界面：I键上一个选项
          K键下一个选项
          回车键选中
          （注意要用小写英文输入）

  游戏内：上下左右移动
          按住z键发射子弹
          x键使用Boom（虽然只能无敌）
          按住Shift进入低速移动模式，显示判定点
